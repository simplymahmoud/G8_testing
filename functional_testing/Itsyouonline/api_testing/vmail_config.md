# Configure verification Email
## 1. Enable IMAP
1. Create **Gmail** email & sign in.
2. From **Settings** > **Forwarding and POP/IMAP** > **IMAP Access**: choose **Enable IMAP**
3. Turn on **Less secure apps**  from
https://www.google.com/settings/security/lesssecureapps

## 2. Create message filter
1. From **Settings** > **Labels** > click on **Create new label**, and set label's name ```itsyouonline```.
2. From **Settings** > **Filters and Blocked Addresses** > click on **Create a new filter**.
2. Put ```ItsYou.Online email verification``` in **subject** field then click on **Create filter with this search**.
3. Check **Apply the label** and from list choose ```itsyouonline``` then click **Create filter**.
